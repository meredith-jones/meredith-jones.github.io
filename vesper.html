Vesper Peak

Home Summary of Hikes About me


Features of this hike:

  • This hike features a couple lakes and stream crossings!
  • The terrain is varied-- forest, rocks, and granite field.
  • The route can be tricky at times. Don't follow orange ribbons. Following cairns can be more helpful, if you're going up a boulder field, you're going the wrong way. Go left.
  • The views from the top are amazing! Go on a clear day!
For more info on this hike, visit wta.org
